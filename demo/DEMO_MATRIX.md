# aw-permission Demo 功能矩阵

| 区域 | 能力 |
|------|------|
| Chip 多选 + 请求所选 | 批量权限 |
| 相机 / 定位 / 存储 / 多权限 | 常用组 |
| Rationale / DSL | `requestWithRationale`、`buildPermissionRequest` |
| 检查 / 设置 | 状态与跳转 |
| 特殊权限 | 悬浮窗、电池、通知（依系统版本） |
| Flow | `observePermissions` 日志 |

README **推荐写法与反模式**、**国产 ROM** 与本 demo 联读。工具栏菜单 **「演示清单」** 可查看本摘要。
